# 5STACK Matchmaking

Sistema fictício de matchmaking para Valorant feito em Python.

O projeto simula a busca de squads com base no rank do jogador, exibindo diferentes times disponíveis para completar uma stack.

## Funcionalidades

- Escolha de rank
- Busca de times por elo
- Geração aleatória de squads
- Sistema de confirmação
- Organização dos ranks em arquivos separados

## Tecnologias

- Python
- Random
- Time

## Objetivo

Projeto criado com foco em prática de lógica de programação, organização de código e desenvolvimento de ideias para o futuro do 5STACK.
